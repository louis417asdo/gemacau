## Cesium Ion token

  camera to Macau.

## Cesium static asset gotcha
Cesium needs its `Workers/`, `Assets/`, `Widgets/` folders served alongside the bundle.
`vite-plugin-cesium` handles this automatically. If you see 404s for cesium workers or a blank
canvas, check that `vite-plugin-cesium` is in `vite.config.js` plugins array.

## Data files
- `data/macau-buildings.geojson` — 11,971 OSM building footprints (4.8MB). Extracted from
  Geofabrik Macau `.osm.pbf` via `scripts/extract-buildings.py`. Only 5.2% have `building:levels`.
- `data/macau-grid.geojson` — 200m × 200m grid with FAR/density pre-computed (679 cells with buildings).
  Generated by `scripts/compute-grid.py`.
- `data/macau-freguesias.geojson` — 8 OSM freguesia boundaries (admin_level=6).
  Generated by `scripts/fetch-freguesias.js`.
- `data/macau-population.json` — DSEC 2021 census population merged from 23 statistical zones into 8 freguesias.
- `data/` is copied to `dist/` during build via `vite-plugin-static-copy`.

## Analysis modules
- `src/analysis/color-scale.js` — logarithmic 6-stop color gradient (dark → green → yellow → orange → red).
  Used by all three analysis layers. Do NOT use linear mapping — it produces all-same-color results.
- `src/analysis/building-density.js` — building footprint density heatmap (200m grid).
- `src/analysis/far-grid.js` — FAR estimation (floor area / cell area, floors are estimated).
- `src/analysis/population-density.js` — population density by freguesia (7 zones, not 23).
- `src/analysis/sunlight.js` — Cesium ShadowMap with summer/winter/equinox presets + time slider.
- `src/analysis/scenario.js` — current vs plan A (green) vs plan B (orange) with extruded rectangles.

## Data honesty (do not overstate to FDCT)
- OSM building heights/floors for Macau: sparse (5.2% have levels, 57.7% have height).
  Remaining buildings use type-based defaults (residential=6, apartments=15, etc.).
- FAR uses grid method (200m cells), NOT official parcel boundaries. Parcel boundaries
  require DSCC cadastre data (non-open, requires agreement).
- Population density is freguesia-level (7 zones), not 23 statistical zones. User plans to
  hand-draw 23 zone boundaries later.
- Sunlight: Cesium ShadowMap is real-time visual shadow, NOT engineering-grade daylight-hours
  analysis. Engineering ray-casting is an M7-M9 task.
- These boundaries are documented in `gemacau_business_plan.md` §三之一 — keep consistent.
